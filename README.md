# 巧克力服铁路线路图系统

一个基于 Supabase + 纯前端的地铁线路图管理系统，支持多维度、环线、坐标辅助等高级功能。

## ✨ 功能特性

### 核心功能
- 🗺️ **多维度地图**：支持主世界、下界、末地、资源世界、异次元五个维度独立管理
- 🚇 **线路管理**：添加/编辑/删除线路，支持环线（首尾相连）
- 🚉 **站点管理**：添加/编辑/删除站点，精确控制 X/Z 坐标
- 🏢 **运营集团管理**：统一管理各线路的运营集团
- 🧭 **路线规划**：基于图的最短路径算法，自动计算换乘站

### 交互功能
- 🖱️ **鼠标/触摸拖拽**：支持鼠标拖拽和手机触摸平移地图
- 🔍 **滚轮缩放**：鼠标滚轮缩放，保持中心点不变
- 🎛️ **缩放控件**：+ / - / 重置 按钮，方便手机操作
- 📍 **点击坐标**：点击地图任意位置显示世界坐标 (X, Z)
- 🧮 **坐标网格**：可开关的辅助网格线（间距自适应）
- 📊 **坐标轴**：可开关的 X 轴（红色）和 Z 轴（蓝色）
- 🎯 **坐标跳转**：输入 X/Z 坐标快速定位到地图某点

### 数据管理
- 💾 **实时保存**：所有修改立即保存到 Supabase 数据库
- 📤 **数据导出**：一键导出所有数据为 JSON 文件
- 📥 **数据导入**：支持导入之前备份的数据（管理员操作）
- 🔄 **维度切换**：不同维度的数据完全隔离，互不影响

### 移动端适配（正在适配！这是画饼）
- 📱 **响应式布局**：手机端自动调整为上下布局
- 👆 **触摸优化**：支持手指拖拽、点击、缩放
- 🎨 **PWA 支持**：可安装到手机桌面
